Manajement Dataset

Aplikasi ini digunakan untuk mengelola dataset yang digunakan pada pemodelan sistem cerdas sehingga dapat ditelusur.

Membuat Tabel Dataset

Membantu pengguna dalam membuat tabel dataset berdasarkan definisi data dari aplikasi Intelligence Creation.

Mengisi / Mengunggah Dataset

Memudahkan pengguna untuk mengunggah dataset yang diperlukan dalam pemodelan perangkat lunak.

Menampilkan Statistik Data

Memberikan analisis statistik deskriptif dan kualitas data untuk memastikan keakuratan dataset.

Merekam Perubahan Dataset

Melacak perubahan dataset yang terjadi akibat pemrosesan data agar dapat ditelusuri dengan mudah.

Merekam Proyek Pengguna

Menunjukkan proyek mana saja yang menggunakan dataset ini untuk meningkatkan transparansi dan keterlacakan.
